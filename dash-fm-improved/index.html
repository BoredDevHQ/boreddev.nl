dash.fm

(AKA dash.fm-improved)

A fork of dash.fm with drastically improved UI and additional widgets. This dashboard displays album art and information about currently playing tracks from Last.fm, integrating data from Wikipedia, Musicbrainz, and Deezer APIs.

Key Features

  • Clean User Interface
  • Real-time track information
  • Stats widget
  • Easy setup page.
  • Open source and free to use.

Introducing: dash.fm

Setup Instructions

Requirements

You need to have PHP installed on your system. Here are the setup instructions for different operating systems:

macOS

php -S localhost:8000

Windows

Using XAMPP:

  1. Download XAMPP from apachefriends.org
  2. Install XAMPP
  3. Start Apache from the XAMPP Control Panel

Or using WSL:

sudo apt update && sudo apt install php php -S localhost:8000

Linux

# Ubuntu/Debian sudo apt update && sudo apt install php php -S localhost:8000

BSD

FreeBSD:

pkg install php php -S localhost:8000

OpenBSD:

pkg_add php php -S localhost:8000

NetBSD:

pkgin install php php -S localhost:8000
